About The Mission

Project Community [name still TBD], is an application meant to be a hub for all things that are social services / community resources. The core reason for developing this application is to give access to as many users as possible to find contact info/locations for the services they need.

At the time of development, the COVID-19 pandemic is ongoing and I have had multiple friends and family that have been effected by COVID-19 and fortunately was able to share the knowledge of these services. Project Community was developed in mind of aiding COVID-19 relief.

In my experience as a caseworker with the state of WA, our mission was to transform lives. Given my background in work with the WA State Department of Social and Health Services, one of the joys was learning about all the social services and resources out there that could help people get back on their feet. If I could not personally help someone with their issue, at the least I could point them in the right direction to get the assistance that they need.

The mission of Project Community is an attempt to continue transforming lives by helping people in a more efficient and larger scale than being an individual caseworker. Although I may not be able to help each and every person personally, Project Community can point people in the right direction and can be a large impact on a person's life.

About The Technology

Project Community is built using Django for the views, routes, models and PostgreSQL to manage the database. Base MVP goals of full CRUD functionality were implemented for full interaction with the database. Multiple stretch goals were reached, key features being:

  • Resource Type Sorting [PostgreSQL / Python / Javascript]
  • Location / Map Data Visualization [Google Maps API]
  • Language Translation Selector [Google Translate]
  • Text/Line-height Vision Accessibility Tools [Javascript / CSS]

Although many stretch goals were reached, the development cycle is never over as improvement is always possible. Future features to look forward to include:

  • Location / Distance sorting
  • Search Functionality
  • Data.gov Resource APIs
  • Third-party log-in [+ User dashboard / Reminders / Bookmarks / Notes]
